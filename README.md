Верстка макета Advance-club

##### Установка пакетов
```sh
$ npm install
```

##### Запуск Gulp - watch
```sh
$ gulp
```

##### Загрузка необходимых библиотек
```sh
$ bower install
```

В проекте используется:
 - [Normalize-css](https://necolas.github.io/normalize.css/)
 - [FontAwesome.io](http://fontawesome.io/)
 
##### Сборка проекта
```sh
$ gulp build
```
